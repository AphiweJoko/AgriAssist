# AgriAssist 🌿

AgriAssist is an AI-powered accessibility tool designed to empower smallholder farmers and agricultural workers by providing intuitive voice interaction, image-based crop recognition, and intelligent text support. This solution bridges the digital divide by enabling users with low digital literacy or impairments to interact with agricultural data systems effortlessly.

## 🔍 Features
- 🎤 Voice Input for seamless interaction
- 🌾 Image Recognition for plant and crop identification
- 💬 Text Assistant for agricultural tips and reminders
- 📱 Responsive frontend using HTML/CSS
- ⚙️ Lightweight backend (Node.js / Flask / other)

## 🌍 Use Case
AgriAssist targets rural farming communities, agricultural extension officers, and accessibility-focused NGOs. It enables more inclusive farming practices through tech-driven, real-world solutions.

## 🚀 Getting Started
1. Clone this repository
2. Install dependencies
3. Run the backend and frontend
4. Use voice or image input to test functionality
